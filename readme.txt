1. 账号必须有人物和存档,此存档必须有两把以上的对局记录(可以使用快速结束比赛),正常获得 1000+VC (重要: 一定要保证VC拿到了)
2. 进游戏后执行程序
3. 按 F12 输入 key(卡密)
4. 按 F10 输入间隔 开始刷VC (最小600秒,建议使用1000秒. 0 = 暂停刷VC)
5. 挂机,等钱到账

(默认开启跳过开包动画)

F5 : 一键结束比赛
F8 : 开始卖收藏 (必须要在梦幻球队界面,输入卖出时间间隔,输入一次卖出的数量)
F10: 开始刷VC ([code:500] 说明 VC 充值成功, 其他值均为失败)
F12: 登录

End: 退出

Ctrl + Del: 清除控制台
Ctrl + Ins: 快速切换账号 (输入0,取消)
Ctrl + Home: 手动修复存档 (输入0,取消)
