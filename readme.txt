1. 账号必须有人物和存档,此存档必须有两把以上的对局记录(可以使用F8快速结束比赛),正常获得 1000+VC (重要: 一定要保证VC拿到了)
2. 进游戏后执行程序
3. 按 F12 输入 key(卡密)
4. 按 F10 输入间隔 开始刷VC (最小1000秒,建议使用1000秒. 0 = 暂停刷VC)
5. 挂机,等钱到账

(打开软件,默认开启跳过开包动画(免费,无需登录))

F5 : 一键结束比赛 (免费,无需登录)
F8 : 开始卖收藏 (一次扣一点,必须要在梦幻球队界面)
F10: 开始刷VC (一次扣一点, [code:500] 说明 VC 充值成功, 其他值均为失败)
F12: 登录

Del: 清除控制台
End: 退出

Ins: 快速切换账号 (待开发)
